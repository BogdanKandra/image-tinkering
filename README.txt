CREATING AN ENVIRONMENT FOR RUNNING THE PROJECT:
=============================
-> Create environment using: "conda create --name image-tinkering python=3.9"
-> Activate environment using: "conda activate image-tinkering"
-> Install pip in the newly created environment: "conda install pip"
-> Get into the project directory, using the "cd" command
-> Install dependencies from requirements.txt: "pip install -r requirements.txt"


RUNNING THE WEB APPLICATION:
=============================
-> From the project directory, get into the "webui" directory
-> Start the Web server using the command: "python app.py"
-> Using a Web browser, visit localhost:8080 for accessing the application
