Modul Imagini (default):
=============================
- Buton in dreapta sus "Change to Video Mode" care permite schimbarea modului de lucru la Video
- Un cadru in care se vor afisa imaginile incarcate (cadru blank pentru 0 imagini)
- O lista de valori:
	- Files Loaded: <Number of files loaded in memory>
	- 0 Files Loaded (If 0 files are in memory)
	- Lista numelor fisierelor incarcate (blank daca 0)
- Buton "Load Files" pentru a deschide fereastra de selectie multipla
	- User-ul selecteaza una sau mai multe imagini din directorul "inputs"; acestea sunt:
		- Incarcate in <img>-uri si afisate (cu File API)
		- Puse intr-o lista de nume care va fi pasata serverului
		- Serverul va cauta fisierele date in directorul "inputs" si le va prelucra
		- Rezultatul / rezultatele sunt salvate intr-un director temp si prezentate clientului. Daca clientul vrea sa salveze, alege un nume si salveaza. Imaginile vor fi salvate in directorul "outputs"
- Buton "Take Selfie" pentru a captura o fotografie prin webcam si a o salva pe disc pentru a putea fi incarcata ulterior.
- Buton "Perform Operation" care deschide un modal cu operatiile posibile pe imagini
	- Checkboxuri cu operatiile, user-ul selecteaza o lista de operatii si o paseaza butonului de "Run"
		- "Keep Red Channel" - Care intoarce codul pentru operatia 'keep_red'
		(Intoarce o imagine formata din canalul rosu al originalului)
	- Butonul "Run" trimite un request la server cu lista de operatii, iar serverul va aplica succesiv operatiile din lista, producand un output
		- Serverul are o ruta gen 'process/op1#op2#opn' care preia operatiile si le aplica succesiv pe imagine(ile) de input si trimita lista de output-uri
	- Butonul "Cancel" care inchide modalul, revenind la meniul principal
	- Outputul este trimis la client, pentru vizualizare, avand apoi doua optiuni:
	"Save" si "Discard". "Discard" sterge din memorie output-ul operatiilor si revine la modalul de selectie. "Save" deschide fereastra de salvare fisier pe disc. Dupa oricare dintre aceste operatii, se inchide modalul.

Modul Video:
=============================
Analog cu Imagini








- Ajax call properties: url, method, data, success, error, dataType (processData, contentType)
- python os functions:
	- os.access() -- determina daca user-ul are drepturi asupra path-ului
	- os.chdir()  -- schimba cwd-ul pe path-ul dat
	- os.getcwd() -- intoarce calea absoluta a directorului curent



- fft, r, g, b:
	- sunt calculate dupa upload
	- serializate la murat intr-o locatie pe server
	- clientul este notificat ca s-au terminat calculele
	- in algoritmi, matricile se vor lua de la murat
	- la al treilea pas, cand user-ul decide daca pastreaza sau discard, se da call la server sa se stearga matricile
- import dinamic
- instantiere dinamica de module (clase)
- apelare dinamica de metoda a unui obiect

- modul Python care contine metode:
	- compute_RGB(image) -- calculeaza matricile R, G, B ale unei imagini si intoarce lista [R, G, B]; daca imaginea nu e RGB, intoarce lista continand imaginea initiala
	- compute_FFT(image) -- calculeaza transformata Fourier a unei imagini; daca imaginea are 3 canale, calculeaza pentru fiecare. Intoarce o lista de transformate Fourier

- la initializare, trebuie sa se calculeze:
	- daca imaginea este grayscale:
		- calculeaza paddedImage
		- FFT pentru paddedImage
		- serializeaza paddedImage FFT
	- daca imaginea este color:
		- calculeaza paddedImage;
		- calculeaza image R, G, B;
		- calculeaza paddedImage R, G, B;
		- FFT pentru paddedImage R, G, B;
		- serializate image R, G, B + paddedImage FFT R, G, B;
			- in tempdata -> image_r, image_g, image_b, image_fft_r, image_fft_g, image_fft_b


- Voi avea clase care fac anumite operatii;
- Fiecare clasa are metode de procesare care iau ca argument o imagine (image) si alti parametrii; deci cand vreau sa aplic o operatie asupra unei imagini, trebuie sa:
	- import dinamic modulul in care se afla operatia (modulul corect e dedus pe baza unui json de configurare);
	- instantiez dinamic clasa care contine operatia;
	- apelez dinamic metoda corespunzatoare a clasei asupra imaginii;

- Cand adaug o operatie noua in proiect, trebuie sa:
	- Adaug o intrare in json-ul de mapare intre nume de operatii si modulul in care se afla
	- Sa fac o clasa noua in care sa pun metoda sau sa o pun intr-o clasa existenta
	- Sa adaug in interfata web buton pentru ea, acolo unde e necesar
